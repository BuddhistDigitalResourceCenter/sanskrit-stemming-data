Instructions
  • These tables demonstrate the various external sandhi transformations.
  • "Finals" represent the final letter(s) in a word.
  • "Initials" represent the first letter(s) in a word.
  • Each table's caption will explain how to recognize which letters transform.
  • Rules with exceptions and notes are marked with an asterisk, and will pop-up when you put the mouse cursor over them.
